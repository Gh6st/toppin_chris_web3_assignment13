Constraints

Affordances as the name implies, afford you to do something if you can. It is the relationship between the user and the object. If the features of the object and the capabilities of the user align, it affords that user to do the task.

Example

My example for affordances is a hook hanger. The upward curve of the hooks affords/indicates that someone can hang things here.
